Privacy Policy

We collect and store up to 30 days the following data on HTTP requests that are made against mapterhorn.com including the subdomains tiles.mapterhorn.com and download.mapterhorn.com:

  • source IP
  • referrer
  • source ASN
  • path
  • domain
  • Cloudflare Point of Presence
  • source browser
  • source operating system
  • source device type
  • source user agent
  • X-Requested-With headers
  • HTTP version
  • cache status
  • edge status code

We may publish aggregate usage statistics such as total requests made, total data served, and similar.

Please revisit this privacy policy regularly. It may change in the future.
